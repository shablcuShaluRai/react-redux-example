# Goals-Todo-App

- This project is for beginners who wants to understand the concept of react and redux.

- In this project, you can add a todolist, remove todolist and toggle todolist(if you will click on that item , that item will be green and line through this item).

- You can also add a goals and you can remove that goals.

In this project we have used following packages
    - React
    - redux
    - react-redux
    - redux-thunk
